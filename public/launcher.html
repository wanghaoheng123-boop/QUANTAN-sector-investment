30-second metric glossary

Hover the ⓘ next to any metric in-app for a longer explanation. Below: the essentials.

RSI(14)

What: momentum oscillator (0–100). Use: >70 overbought, <30 oversold; persistent >70 in strong trend = continuation.

MACD

What: EMA12 − EMA26 with signal-line crossover. Use: bullish on positive crossover; rising histogram = trend strengthening.

EMA200 / 200SMA

What: long-term trend filter. Use: most quant systems require price > EMA200 for new longs.

ATR(14)

What: avg daily range (volatility). Use: set stops at 1.5×ATR; size positions inversely to ATR%.

Sharpe Ratio

What: annual return ÷ volatility. Use: >1 good, >2 excellent. Better metric than raw return.

Sortino Ratio

What: Sharpe but only penalises downside. (Phase 12 fix: correct denominator n_d, count of negative periods.) Use: >1.5 = good downside management.

Profit Factor

What: gross profit ÷ gross loss. Use: >1.5 robust, >2 excellent. <1.3 = fragile.

Max Drawdown

What: largest peak-to-trough loss. Use: pre-approve mental tolerance; if live DD > 1.5× backtest DD, model is broken.

GEX (Gamma Exposure)

What: dealer gamma at each strike. Use: high positive = pinning; deeply negative = destabilising flow, expect bigger moves.

Dark Pool Prints

What: off-exchange institutional block trades. Use: clusters near support/resistance carry signal — but rarely same-day.

Regime (BTC / Sector)

What: discrete state (Strong Bull / Bull / Neutral / Bear / Strong Bear / Euphoria / Capitulation). Use: match strategy to regime.

Data Freshness

What: pulsing dot showing data age. Use: green = live, amber = recent (10–120s), red = stale (>2min) — refresh before acting.

Decision workflow — turn signals into action

Same disciplined sequence regardless of which dashboard you start in.

  1. Spot. Use Sector Home / Heatmap / Briefs to find what's moving.
  2. Check regime. Is the broader sector / market bullish (price > EMA200)? If no, skip longs.
  3. Drill in. Open the Sector or Stock detail page. Look for indicator alignment: trend (EMA200) + momentum (RSI 50–70) + MACD positive crossover.
  4. Validate via backtest. Has this signal pattern actually paid in this name historically? Open Backtest dashboard.
  5. Size correctly. Place a 1.5×ATR stop. Position size so hitting that stop = ≤1% of account.
  6. Set alerts, not orders. Wait for confirmation in the next session before adding.
